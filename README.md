## Typography

- **Primary Font:** Bold condensed sans-serif (headlines)
- **Body Font:** Clean sans-serif (Calibri / Arial)
- **Style:** High contrast, dark backgrounds, strong hierarchy

## Core Frameworks

### The Mission5 Framework™
Identity → Purpose → Goals → Routines → Habits

*"You are the common denominator. Build the foundation first."*

### The 5 Business Phases
**Founder → Build → Grow → Scale → Tycoon**

The goal: creating separation between time invested and money earned — true lifestyle freedom.

| Phase | Where You Are | EHR Benchmark |
|-------|--------------|---------------|
| Founder | Starting from scratch | $0–30/hr |
| Build | $5K–$15K/month revenue | $70/hr to exit |
| Grow | Business is stable, work smarter | $200/hr |
| Scale | Business runs without you | $750/hr to exit |
| Tycoon | Assets work for you | $750+/hr |

### The Ascent Programs

| Program | Format | Description |
|---------|--------|-------------|
| **SUMMIT** | 1-on-1 Coaching | Faster ascent, premium support |
| **ASCENT** | Group Coaching | Community support, steady progress |
